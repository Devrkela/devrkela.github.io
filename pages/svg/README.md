# Objectif

L'objectif était de créer un module permettant de calculer le point proche sur une courbe de bézier, par rapport à un autre point.

# Stack

HTML, CSS, Javascript et SVG.

# Difficultés

- Comprendre les formules mathématiques derriére les courbes de Bézier.
- Définir et effectuer le calculs pour réaliser l'objectif.
- Mettre en place la solution.
