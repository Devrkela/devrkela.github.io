# Objectif

L'objectif était de créer un module permettant de rechercher une ou des fréquence(s) d'un signal périodique, en utilisant une transformée de Fourier.

# Stack

HTML, CSS, Javascript et SVG.

# Difficultés

- Comprendre comment retranscrire les formules mathématiques en code.
- Mettre en place l'interface SVG.
