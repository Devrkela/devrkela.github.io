# Objectif

L'objectif était de créer un module permettant d'utiliser le GPU afin de manipuler les espaces colorimétriques XYZ et LUV.

# Stack

HTML, CSS, Javascript et WebGL2.

# Difficultés

- Découvrir et manipuler l'API WebGL2.
- Écrire un fragment shader spécifique.
- Comprendre l'utilité d'un espace colorimétrique différent de RGB.
