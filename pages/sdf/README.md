# Objectif

L'objectif était de créer un module permettant de générer une image 3D, en utilisant une technologie de Ray Marching.

# Stack

HTML, CSS, Javascript et WebGL2.

# Difficultés

- Comprendre comment un Ray Marcher fonctionne.
- Comprendre comment l'API WebGL2 fonctionne.
- Mettre en place le Ray Marcher.
