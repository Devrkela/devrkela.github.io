Référence

Cette page contient les différents outils implémentés par Ascension.

API

alimente(url: string, type?: DictionnaireType): function

Effectue une requête de la ressource et retourne un objet spécifique selon le type.

Retourne:

  • La valeur retournée par la fonction, si le type est iife ou si il n'y a pas de type.
  • Un HTMLStyleElement, si le type est css.
  • Un Document, si le type est html.
  • Un HTMLImageElement, si le type est image.
  • Un objet littéral, si le type est json.
  • Un SVGElement, si le type est svg.
  • Un DocumentFragment, si le type est template.
  • Un HTMLVideoElement, si le type est video.

entrepose{put: function, get: function}: object

Facilite la mise en cache des ressources.

put(url: string, value: any, type?: DictionnaireType)

Ajoute la ressource dans le cache.

get(url: string, type?: DictionnaireType)

Récupére la ressource du cache.

Retourne la même chose que la fonction alimente.

distribue(url: string, type?: DictionnaireType)

Récupére une ressource via le réseau ou le cache, l'entrepose, si besoin, et retourne la ressource.

Retourne la même chose que la fonction alimente.

fabrique(url: string, options?: OptionsComposant)

Récupére un composant. L'url doit pointer vers un dossier contenant un fichier "model.html", "view.css" et "controller.js". Les ressources sont automatiquement entreposées.

Le paramètre options permet de filter les fichier recherchés dans le dossier. Par exemple, si html vaut false, la fonction ne cherchera pas la ressource "model.html".

Retourne un objet du type Composant.

installe(composant: Composant)

Installe un composant dans le document.

Interface

DictionnaireType: Set<String>

Représente un type de ressource.

La type doit être une valeur des valeurs suivantes {"iife", "css", "html", "image", "json", "svg", "template", "video"}

OptionsComposant: Object

Représente un objet permettant de filtrer les fichiers qui composent le composant.

L'interface de l'objet est {html?: boolean, css?: boolean, js?: boolean}

Composant: Object

Représente les ressources inclus dans le composant.

L'interface de l'objet est {html?: HTMLElement, css?: HTMLElement, js?: function}
